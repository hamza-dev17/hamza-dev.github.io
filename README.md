# Project Aether

A small gallery of single-file HTML visual experiments and interactive art pieces.

## Live Structure
- `index.html` - gallery homepage
- `projects/index.html` - gallery mirror for the project section
- `projects/burst/` - interactive hand-tracking particles
- `projects/hidden-image/` - liquid reveal effect
- `projects/liquid/` - fluid mirror simulation
- `projects/tear/` - holographic celestial gate

## Notes
- Some pages use webcam access and hand tracking.
- If a browser blocks camera access, allow permissions for the page.
- Old top-level filenames are kept as redirects for convenience.
